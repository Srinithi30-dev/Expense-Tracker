# Expense AI — Smart Spending Intelligence

Ever wondered where your money actually goes every month?

Expense AI is a smart, AI-powered expense analysis system that helps you **understand your spending patterns, detect habits, and make better financial decisions** — all from your transaction data.

##  What This Project Does

Instead of just tracking expenses, this system:

-  Accepts transaction data (GPay, PhonePe, bank exports)
-  Automatically classifies expenses into categories
-  Visualizes spending in an intuitive dashboard
-  Shows trends and patterns over time
-  Provides insights to improve spending habits

---

##  Why This Matters

Most expense trackers just show numbers.

This project goes a step further:
> It helps you **think about your spending**

For example:
- Are you overspending on food delivery?
- Do weekends increase your expenses?
- Which payment mode do you rely on most?

---

## How It Works

1. User uploads a CSV file
2. Backend processes data using Python
3. Transactions are categorized using intelligent logic
4. Data is analyzed and sent to frontend
5. Dashboard visualizes everything in real-time
